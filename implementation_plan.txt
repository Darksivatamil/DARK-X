## How to Use These Plans
## Project Overview
Each part below is a **self-contained build plan**. You can give any part to an AI (or build it yourself) and it will produce working code. Build them **in order** — each part depends on the previous ones.
DARK-X is a full-stack web application that combines:
- 🤖 **13+ AI Agents** — each with unique names, personalities, and capabilities
- 🎮 **Gamified Skill Progression** — Solo Leveling-inspired leveling system with 100 unique powers
- 📡 **Live Security Modules** — Network scanning, traffic analysis, fuzzing, and more
- 📰 **Daily Content** — Hacking news, AI updates, tool guides, and coding challenges
- 🖥️ **Premium Dark UI** — Anime-inspired dashboard with electric effects, particles, and animations
- 💬 **DARK-GUIDE-X Chat Agent** — Full DARK-GUIDE skill integrated as a chat companion
- 🔐 **Auth System** — Username/password login with session management
- ⌨️ **Command Terminal** — `/arise`, `/rest`, `/refresh`, `/logout` and agent commands
> [!TIP]
> **Workflow**: Open Part 1 → give it to an AI → verify it works → move to Part 2 → repeat until Part 8 is done → DARK-X is live.

- `api_keys` (JSON — stores OpenAI, Gemini keys)

### [NEW] backend/models/progress.py
Progress tracking:
- `user_id`, `overall_level`, `total_xp`
- `python_level`, `python_xp`
- `kali_level`, `kali_xp`
- `ai_level`, `ai_xp`
- `industry_level`, `industry_xp`
- `powers_unlocked` (JSON array of power IDs)
- `current_rank` (e.g., "Shadow Initiate", "Cyber Hunter", etc.)
- `streak_days`, `tasks_completed`, `tasks_submitted`
### [NEW] backend/routers/auth.py
Endpoints:
- `POST /api/auth/register` — Create account (username + password)
- `POST /api/auth/login` — Login, returns JWT token, records timestamp
- `POST /api/auth/logout` — Logout, updates session stats
- `GET /api/auth/me` — Get current user profile
- `PUT /api/auth/settings` — Update API keys and preferences

### [NEW] backend/agents/base_agent.py
Base agent class providing:
```python
class BaseAgent:
    name: str              # Agent code name
    full_name: str         # Display name  
    role: str              # Brief role description
    personality: str       # Personality traits for AI prompts
    capabilities: list     # List of things this agent can do
    is_active: bool        # Active/dormant state
    status: str            # "dormant" | "active" | "processing"
    memory: dict           # Persistent memory per user
    offline_capable: bool  # Can work without internet?
    
    async def activate(user_id)      # Wake up with intro message
    async def deactivate()           # Go dormant
    async def process(message, user) # Handle user interaction
    async def get_intro()            # Self-introduction
    async def remember(key, value)   # Store to memory
    async def recall(key)            # Retrieve from memory
```
