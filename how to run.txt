DARK-X — HOW TO RUN
====================

1. Open terminal in this directory
2. Run: uvicorn backend.main:app --host 0.0.0.0 --port 8081
3. Open browser to: http://localhost:8081/static/index.html
4. Register an account and login

FEATURES
- Full 8-tab dashboard with real API data
- All 15 agents with chat functionality
- WebSocket terminal with /arise, /rest, /help commands
- 10 security modules with realistic output
- Gamification: 100 powers to unlock across 100 levels
- 8 ranks from Shadow Initiate to Shadow Monarch
- Export: PDF reports and JSON backup
- Ctrl+K: Search bar / Ctrl+`: Terminal toggle
